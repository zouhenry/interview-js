    What's the difference between ng-options and ng-repeat?
    Which one is preferred and why?

    //ng-repeat
    <select ng-change='...'>
      <option ng-repeat='option in options' value='option'>{ { option } }<option>
    </select>

    //ng-option
    <select ng-model='...' ng-option='option in options'> <option>
